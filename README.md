# **飛行步驟**

**非常重要:**

*解鎖無人機前，一定要確保無人機旁沒有人 !!!! 兩隻拇指都需要放在搖桿上。 *

1. 拉起機臂和確認裝上螺旋槳和無人機電池即操控器電池。

2. 開啟操控器(中間按鈕上推)和將無人機接上電池(**閃黃燈-初始化**)。

3. 飛控板旁有按鈕，長按解鎖 (**解鎖會亮藍燈**)。

4. 確定是藍燈後，操控器左手*往下推再向右*即**解鎖**。

5. 切成定位模式時，沒有連接到GPS會有音階音效，連接到GPS會逼一聲。

6. 左手往上輕推油門，可以透過右手調整。

7. 降落時，左手**慢慢**往下推，*下推到底往左*即**上鎖**。
